# Rural_Health_Hospital_Sys_Java
A full-stack role-based Hospital Management System developed using Java (Servlet &amp; JSP), MySQL, HTML, and CSS. This project manages core hospital operations including doctor approval workflow, patient login/registration, and appointment booking.


##### Rural Hospital Management System   #######

A web-based Hospital Management System developed using HTML, CSS, Java, JSP, Servlet, and MySQL.

##  Features

###  Admin
- Approve/Reject Doctor Requests
- View Patients
- Manage Appointments

###  Doctor
- Login
- View Appointments
- Change Password

###  Patient
- Register & Login
- Book Appointment
- Change Password

##  Technologies Used
- Java (Servlet & JSP)
- MySQL
- HTML, CSS
- Apache Tomcat

##  Database
Import the provided hospitaldb.sql file into MySQL.
